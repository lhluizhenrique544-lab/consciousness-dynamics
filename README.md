$$dC/dt = (A \cdot |s| - k \cdot N \cdot C) / f_s$$

Where:
- **A**: Environmental Gain (Stimulus intensity)
- **N**: Homeostatic Resistance (Dissipation)
- **s**: Input signal (Normalized)
